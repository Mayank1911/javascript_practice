BOM Objects

BOM stands for Browser Object Model

It's a model which tends towards the relation between Javascript and web browser


OR

It is a set of objects that represent the browser window and the document included in it. It alsocontains objects that represent the browser's history of visited pages, the geolocation of the browser, and the user's clipboard.

Types of BOM Objects

  • Window object
  • Navigation object
  • History object
  • Location object
  • Screen object
